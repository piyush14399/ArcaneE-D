File Encryption and Decryption GUI App with Python

Tech Used-
Python 3
Tkinter for GUI
pycryptodomex for Cryptodome AES Encryption
cx_freeze to build executable

Other Installtions Required- 
pycryptodomex for AES encryption - "pip install pycryptodomex"
cx_freeze to build executable - "pip install cx_freez"

To Build Executable .exe file- 
Open setup.py and replace <path/to/your/python_directory> with your python installation directory.
Open Terminal in directory where setup.py is; and type -
python setup.py build
